# GramPulse AI

## AI-Driven Cash Flow Prediction & Risk Flagging System for Rural Micro Enterprises

GramPulse AI is an AI-powered financial intelligence platform designed to predict cash flow and identify financial risks for rural micro-enterprises.

## Features

- Cash flow forecasting
- Risk flagging
- Climate and market intelligence
- Explainable AI
- Farmer dashboard
- Field officer dashboard

## Technology Stack

- Python
- FastAPI
- Streamlit
- XGBoost
- SHAP
- PostgreSQL

## Repository Structure

- docs/ : Project documents and presentations
- data/ : Datasets
- backend/ : API code
- frontend/ : Dashboard
- models/ : ML models
